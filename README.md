## Engineering principles

1. **Deterministic core, model edge.** Language is useful input, not authority.
2. **Evidence before action.** Consequential state changes need enforceable checks.
3. **Failure is designed.** Finite retries, idempotency, health, rollback, and
   backup evidence belong in the architecture.
